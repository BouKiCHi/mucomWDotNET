# mucom.NET
  mucom88の.NET版です。コンパイラのみ。  
  
[概要]  
　mucom88を.NET版に移植したものです。コンパイラのみ。  
  
[機能、特徴]  
 ・恐らく生データの限界である64Kbyteフルに使用したmubが作成できます。未確認。  
  
[必要な環境]  
 ・Windows7以降のOSがインストールされたPC  
 ・テキストエディタ  
 ・.mubを演奏できるプレイヤアプリ  
 ・気合と根性
